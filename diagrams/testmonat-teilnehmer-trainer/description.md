# Testmonat-Teilnehmer durch den Trainer

**Type:** Flowchart — Trainer role
**File:** `Testmonat-Teilnehmer durch den Trainer.png`

Trainer logs in, opens the training overview (filtered to sessions with trial-month students), selects a training, and views the registered trial students. For each student the trainer can: mark attendance or absence, optionally add a comment on the training, optionally add a comment on the student.
