# User Flow Registrierung Trainingsangebot

**Type:** Flowchart — Prospective student role
**File:** `User Flow Registrierung Trainingsangebot.png`

Prospective student visits the training page, selects an offer, and fills the signup form. If their email is new → registration email sent → they fill and submit the contract form → admin is notified. If the email already exists → status email about the trial month is sent directly.
