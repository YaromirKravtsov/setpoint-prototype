# Bearbeitung des Testmonats-Antrags

**Type:** Flowchart — Admin role
**File:** `Bearbeitung des Testmonats-Antrags.png`

Admin receives a trial month application via email, opens the system, and reviews the student's data. Decision: accept → assign to a training group, set status to "Student in trial month", send welcome email. Reject → add a comment and decline.
